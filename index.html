Jason's Elements HTML Cheetsheet

Element

Description
<li> The <li> element let's you create 2 types of list <ol> and <ul>
<ol> <ol> is used inside of the <li> element to create an ordered list that is in numbered order
<ul> Used to create an unordered list in bullet points inside the <li> element
<video> <video> is used to embed a video onto your webpage
<em> Used to create emphasis on a piece of text in your code usually in the form of italics
<div> The <div> element is used as a container to divide up your code into sections to make it easier to manage
<br> The break tag is used to seperate lines of text as changing text onto a sperate line in your code will not put your text on a seperate line
<img> Used to embed an image onto your webpage. This is a self closing element so theres no need for a closing tag
<p> The paragraph element is used to display a block of text
<body> Content inside of the <body> element are displayed on your webpage (there can only be one body tag in your html code)
<span> Used as an inline container for text that can be grouped for styling purposes
<strong> The <strong> element is used as an container for important or urgent text and is usually rendered by the browser as bold
<a> The anchor element is used to create hyperlinks in the HTML document. the hyperlink can point to other webpages, files on the same server, a location on the same page, or any other URL via the hyperlink
<head> the head element contains general information about an HTML page that isn't displayed on the webpage itself. This can include stylesheets and font types.
<title> The title element contains a text that defines the title of an HTML document. The title is displayed in the browser's title bar or tab in which the HTML page is displayed. The title element can only be contained inside a document's head element.
